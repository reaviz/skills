# Common

Shared infrastructure components used across all reaviz charts. These modules handle axes, gridlines, tooltips, legends, visual effects, interactions, and color management.

## Components

| Component | Description |
|-----------|-------------|
| **Axis** | LinearXAxis, LinearYAxis — tick formatting, labels, positioning |
| **Gridline** | GridlineSeries, Gridline, GridStripe — background grid patterns |
| **Tooltip** | ChartTooltip, TooltipArea, TooltipTemplate — hover data display |
| **Legend** | DiscreteLegend, SequentialLegend — series identification |
| **Gradient & Mask** | Gradient, GradientStop, RadialGradient, Mask, Stripes — SVG fills |
| **Brush & ZoomPan** | ChartBrush, ChartZoomPan — interactive range/zoom |
| **Color Schemes** | Named palettes, ColorSchemeType — consistent color management |
| **Markers** | LinearValueMarker, RadialValueMarker, MarkLine — reference lines |
