# Interaction

User interaction modules: selection, camera controls, context menus, and node collapsing.

## Components

| Component | Description |
|-----------|-------------|
| **Selection** | useSelection hook, lasso, single/multi selection, path selection |
| **Camera** | CameraControls, useCameraControls, useCenterGraph, camera modes |
| **Context Menu** | RadialMenu with slices for node/edge right-click |
| **Collapse** | useCollapse hook for expand/collapse node hierarchies |
